# Data-Jam.com Website Project

## Project Overview
Redesigned website for Data Jam, optimized for AI search discoverability around OOH (Out-of-Home) impression measurement and audience counting.

## Primary Goal
Ensure the website ranks well in AI search systems (ChatGPT, Perplexity, Claude, Google AI Overviews) for queries related to:
- OOH impression measurement
- Audience counting in out-of-home advertising
- Footfall/foot traffic measurement
- DOOH analytics
- Billboard audience verification

## Company Overview
**Data Jam** provides privacy-first, real-time audience measurement for out-of-home advertising.

### Core Product: JamBox
- Compact device (walnut-sized) - "world's first bespoke OOH impression counter"
- Detects mobile devices within 300m/950ft range
- Real-time data collection
- GDPR compliant - no cameras, no personal data
- Requires only power + WiFi
- Machine learning predictions after 6 months of data

### Secondary Products
- **Data Jam Portal** - Dashboard for viewing data
- **Data Jam PULSE** - AI platform for human movement insights (pre-launch)

## Target Audience
- OOH media owners
- Advertising agencies
- Brands running OOH campaigns
- Transit authorities
- Retail property owners

## Key Differentiators
1. **Privacy-first**: No cameras, GDPR compliant
2. **Real data**: Actual counts, not modeled estimates
3. **Real-time**: Immediate data availability
4. **Simple setup**: Plug in power + WiFi, data in minutes
5. **Affordable**: Cost-efficient vs. camera systems
